# IT 行业动态简报 - 2026 年 03 月 06 日

## 🔥 头条要闻

### 1. GPT-5.4 正式发布
OpenAI 发布了 GPT-5.4 版本，继续推进大模型能力边界。该版本在 Hacker News 上引发热议，目前已有 683 点投票和近 600 条评论。
- **来源**: OpenAI 官方博客
- **热度**: HN 683 points

### 2. Epic v. Google 退出最高法院诉讼
Epic Games 与谷歌双方同意撤回提交给美国最高法院的上诉。根据最高法院规则 46.1，如果所有当事方书面要求驳回案件，书记员将直接输入驳回令。这意味着这场持续多年的应用商店反垄断诉讼将在下级法院层面继续。
- **来源**: The Verge / 美国最高法院文件
- **影响**: 谷歌应用商店政策可能面临新的调整

### 3. Wikipedia 遭遇大规模管理员账户入侵
Wikipedia 被迫进入只读模式，原因是大量管理员账户被攻破。这是维基百科历史上罕见的安全事件，目前仍在调查中。
- **来源**: Wikimedia Status
- **热度**: HN 907 points

## 💻 开发者工具与技术

### Chrome 144 引入 Temporal API
Chrome 144 版本正式引入突破性的 Temporal API，为 JavaScript 中的日期和时间管理带来革命性变化。作为饱受诟病的 Date 对象的现代化替代方案，Temporal 解决了解析歧义、时区处理复杂以及可变算术运算等问题。
- **来源**: InfoQ
- **状态**: 不同浏览器支持程度仍有差异

### 谷歌开发者文档进入 AI 智能体时代
谷歌宣布 Developer Knowledge API 公开预览版，附带模型上下文协议（MCP）服务器。这为 AI 开发工具提供了简单、机器可读的方式来访问谷歌官方开发者文档。
- **来源**: InfoQ
- **意义**: AI 编码助手将能更准确获取官方文档

### Warper 7.x 发布：Rust 驱动的 React 虚拟化库
Warper 发布 7.x 版本，带来性能优化、改进的打包工具兼容性以及更完善的开发者工具支持。该库由 Rust 和 WebAssembly 驱动。
- **来源**: InfoQ

### GitHub Issue 标题攻击影响 4000+ 开发者机器
安全研究人员发现，通过精心构造的 GitHub Issue 标题，攻击者可以影响使用特定 AI 编码工具的开发者机器。该漏洞被称为"Clinejection"。
- **来源**: Grith.ai
- **热度**: HN 348 points

## 🔐 安全与隐私

### FBI 和 Europol 关闭 LeakBase 黑客论坛
美国执法部门与 Europol 合作，关闭了拥有超过 14.2 万名成员的 LeakBase 黑客论坛。该论坛数据库包含"数亿"条被盗账户凭证。
- **来源**: The Verge / 美国司法部
- **行动**: 全球联合执法行动

### CBP 利用在线广告生态系统追踪人员移动
调查显示，美国海关和边境保护局（CBP） tapped into 在线广告生态系统来追踪人员移动。这一做法引发隐私权倡导者的强烈关注。
- **来源**: 404 Media
- **热度**: HN 396 points

### Proton Mail 协助 FBI 识别匿名抗议者
法庭记录显示，Proton Mail 响应瑞士当局的请求，提供了与"Stop Cop City"抗议活动相关账户的支付信息，随后这些信息被转交给 FBI。尽管 Proton 提供端到端加密，但支付元数据较难隐藏。
- **来源**: 404 Media / The Verge

### Firefox 崩溃 10% 由 bitflips 引起
Mozilla 工程师发现，Firefox 浏览器约 10% 的崩溃是由内存位翻转（bitflips）引起的。这一发现对软件可靠性研究有重要意义。
- **来源**: Mastodon / HN
- **热度**: HN 349 points
