# IT 行业动态简报 - 2026 年 03 月 07 日

## 🔧 开发技术

### Kubernetes 引入节点就绪控制器
Kubernetes 项目最近宣布了一个名为**节点就绪控制器（Node Readiness Controller）**的新核心控制器，旨在通过使 API 服务器对节点就绪状态的视图更加准确，从而增强调度的可靠性和集群健康。
*来源：InfoQ*

### Chrome 144 引入 Temporal API
Chrome 144 版本正式引入突破性的**Temporal API**，为 JavaScript 中的日期和时间管理带来了革命性变化。作为饱受诟病的 Date 对象的现代化替代方案，Temporal 解决了解析歧义、时区处理复杂以及可变算术运算等问题。
*来源：InfoQ*

### Warper 7.x 发布：Rust 驱动的 React 虚拟化库
Warper 是一个由 Rust 和 WebAssembly 驱动的开源 React 虚拟化库，近日发布了 7.x 版本，带来性能优化、改进的打包工具兼容性以及更完善的开发者工具支持。
*来源：InfoQ*

## 🤖 人工智能

### 谷歌开发者知识 API 公开预览
谷歌宣布了**开发者知识 API（Developer Knowledge API）**的公开预览版。它附带了一个模型上下文协议（MCP）服务器，为 AI 开发工具提供了一个简单、机器可读的方式来访问谷歌的官方开发者文档。
*来源：InfoQ*

### OpenAI 推迟 ChatGPT"成人模式"
OpenAI 发言人表示，原计划本季度推出的 ChatGPT"成人模式"将被推迟，以便团队能够专注于更高优先级的功能，包括智能提升、个性改进、个性化以及让体验更加主动。
*来源：The Verge*

### OpenAI Codex 安全功能上线
OpenAI 推出了**Codex Security**研究预览版，这是一个专注于识别和修复应用程序安全问题的 AI 代理。同时，Codex 开源基金现在为开源开发者提供为期六个月的 ChatGPT Pro with Codex 订阅。
*来源：The Verge*

## 📊 行业观察

### 加州社区学院 AI 聊天机器人效果不佳
三所加州社区学院每年花费高达 50 万美元购买 AI 聊天机器人，用于帮助学生解答经济援助和招生问题。但 CalMatters 报道称，这些机器人"能正确回答一般性问题，但在处理更具体问题时表现不佳"，甚至有学校的机器人无法正确说出自己校长的名字。
*来源：The Verge*

### 企业管理反思：95% 的加班是形式主义
晚点采访中指出，企业经营策略上的"内卷"是指业务竞争，而非形式主义加班。95% 以上的加班都是没有意义的，守在工位、搞多余报表都是无效劳动。
*来源：晚点 LatePost*
