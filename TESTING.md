# 网站发布测试清单

**创建时间**: 2026-03-03 08:50
**目的**: 确保每次发布前都经过完整测试

---

## 📋 测试清单

### 1. 构建测试
- [ ] `npm run build` 无错误
- [ ] 所有页面生成成功
- [ ] 构建产物大小合理

### 2. 本地预览测试
- [ ] `npm run preview` 启动正常
- [ ] 首页加载正常
- [ ] 文章详情页加载正常
- [ ] 导航菜单功能正常
- [ ] 分页功能正常
- [ ] 分类筛选功能正常

### 3. 部署测试
- [ ] Cloudflare Pages 部署成功
- [ ] 预览 URL 可访问
- [ ] 所有页面 HTTP 状态码 200
- [ ] 图片等静态资源加载正常

### 4. 功能测试
- [ ] 导航菜单点击跳转正确
- [ ] 分类筛选显示正确文章
- [ ] 分页上一页/下一页正常
- [ ] 页码点击跳转正确
- [ ] 文章链接正确

### 5. 兼容性测试
- [ ] Chrome 浏览器正常
- [ ] Safari 浏览器正常
- [ ] Firefox 浏览器正常（可选）
- [ ] 移动端响应式正常

### 6. 性能测试
- [ ] 首页加载时间 < 2 秒
- [ ] 图片压缩合理
- [ ] 无控制台错误

### 7. 内容验证
- [ ] 所有文章标题正确
- [ ] 文章分类标签正确
- [ ] 日期显示正确
- [ ] 文章描述正确

### 8. SEO 测试
- [ ] 页面 title 正确
- [ ] meta description 正确
- [ ] GA4 统计代码加载
